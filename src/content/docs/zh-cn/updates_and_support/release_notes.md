## [v1.0.0.dev2](https://pypi.org/project/pygamestudio/1.0.0.dev2/)
### 新增功能
- 新增桌面应用一键打包功能，可将项目打包为独立可执行文件，其他用户无需安装 Python 环境即可直接运行游戏
- 新增线条对象，丰富场景编辑元素
- 新增移动 Gizmo 控制器，支持可视化拖拽移动场景对象
- 新增 `get_object_by_path` 接口，支持通过路径精准获取场景中的指定对象
- 帮助菜单栏新增更新日志入口，方便查看版本迭代记录

### 优化改进
- 优化新建项目窗口布局，提升操作交互体验
- 优化项目模板中 `main.py` 初始代码，结构更规范、易用性更强

### 问题修复
- 修复部分窗口鼠标指针样式显示错误问题
- 修复 Dashboard 面板项目删除失败的 Bug

## [v1.0.0.dev1](https://pypi.org/project/pygamestudio/1.0.0.dev1/)
### 新增功能
- 完成基础可视化编辑界面搭建，包含层级窗口、资源管理窗口、场景编辑窗口、日志输出窗口、属性配置窗口等核心面板
- 支持创建矩形、椭圆、文本三种基础可编辑对象
- 实现 Windows、macOS、Linux 三大操作系统平台支持
- 新增项目创建、管理功能，支持独立项目工程化管理
- 集成实机运行预览功能，可实时查看项目运行效果
- 新增界面语种切换功能，支持中文简体与英文自由切换
